# Simple To Do List

This is a simple and clean To-do list website that does it's job. It is built using HTML, CSS, and Javascript and features a white and pastel pink theme. The app allowa users to quickly add and remove tasks, making it useful for managing reminders and small tasks. The layout is lightweight, quick to respond, and easy to navigate. All it has are two buttons where you can add or delete a task if you wish to do so.
